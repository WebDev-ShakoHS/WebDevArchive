If you are still in school...

you should focus on taking as many computer classes as possible, even if the topic of the class is not what you are interested in. For example, even if you do not want to make a website, taking a class like Web Development could still help a lot because you learn general things about computers and it gets your brain thinking like a computer.

If you are done with school...

I would recommend finding what you specifically want to do with computers and focus on the basics of that topic. Using youtube could very much help with that but there are plenty of website out there for pretty much anything. For example, if you are interested in coding you could use Code Academy.

What do these skills give me?

Knowing how to use computers can help you in an infinite number of ways. An obvious one would be that it could get you a job, but it can also help with solving errors with your own computer, you could do something like make a website for someone if they need one, and many other things. As the world progresses, computers are becoming more and more important to society, so having a general idea of how computers work definitely would not hurt.
